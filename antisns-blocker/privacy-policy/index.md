# プライバシーポリシー

最終更新日：2026年5月18日

本プライバシーポリシーは、「SNS断ちアプリ」（以下「本アプリ」）における個人情報の取り扱いについて説明します。

## 1. 収集する情報

本アプリは、**個人情報を収集しません**。

本アプリが端末内に保存するデータは以下のみです。

- ブロック対象として選択したアプリの情報（端末内の UserDefaults にのみ保存）
- ブロックの有効・無効の状態（端末内の UserDefaults にのみ保存）

これらのデータは端末の外部に送信されることはなく、開発者を含む第三者がアクセスすることはできません。

## 2. 第三者への情報提供

本アプリは、いかなる情報も第三者に提供・販売・共有しません。

## 3. 外部サービスの利用

本アプリは外部のサーバー、アナリティクスサービス、広告ネットワークを一切使用しません。

## 4. スクリーンタイム API の利用について

本アプリは Apple の FamilyControls フレームワーク（Screen Time API）を使用して、ユーザー自身が選択したアプリへのアクセスを制限します。この機能はユーザー本人の操作によってのみ機能し、取得した情報が端末外に送信されることはありません。

## 5. お子様のプライバシー

本アプリは13歳未満のお子様を対象としておらず、意図的に13歳未満の方の個人情報を収集することはありません。

## 6. プライバシーポリシーの変更

本ポリシーを変更する場合は、このページを更新します。重要な変更がある場合はアプリのアップデートを通じてお知らせします。
